# Game Design Document (GDD) Master Template

This directory contains a comprehensive GDD template synthesized from best practices of top-tier studios like Voodoo, miHoYo (Genshin Impact), and Tencent.
Each chapter is a standalone Markdown file, suitable for integration into Wikis or documentation systems.

- **[1_Copyright Information.md](1_Copyright%20Information.md)** – Copyright and Legal Information
- **[2_Version History.md](2_Version%20History.md)** – Version Control and Approval Records
- **[3_Game Overview.md](3_Game%20Overview.md)** – Game Overview and Product Definition
- **[4_Gameplay and Mechanics.md](4_Gameplay%20and%20Mechanics.md)** – Core Gameplay and System Mechanics
- **[5_Story, Setting and Character.md](5_Story,%20Setting%20and%20Character.md)** – World Building and Character Design
- **[6_Levels.md](6_Levels.md)** – Level Design and Content Structure
- **[7_Interface.md](7_Interface.md)** – UI/UX System
- **[8_Artificial Intelligence.md](8_Artificial%20Intelligence.md)** – AI Design
- **[9_Technical.md](9_Technical.md)** – Technical Architecture and Solutions
- **[10_Game Art.md](10_Game%20Art.md)** – Art Style and Asset Planning
- **[11_Secondary Software.md](11_Secondary%20Software.md)** – Tools and Secondary Software
- **[12_Management.md](12_Management.md)** – Project Management and Operations
- **[13_Appendices.md](13_Appendices.md)** – Appendices and Asset Lists

## GDD Deliverable Structure Guide
- [GDD_Deliverable_Structure.md](../../templates/guides/gdd_deliverable_structure.md) – Standard directory structure for GDD deliverables.
