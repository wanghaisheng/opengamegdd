# 11. Secondary Software

> Describe the supporting software for game development, publishing, and operations, such as editors, installers, and updaters.

## 11.1 Editor

- **Level Editor:**
    - **Scope:** Terrain modification, object placement, enemy wave configuration, and quest scripting.
    - **Users:** Level Designers, System Designers, Technical Artists.
    - **Output:** Data format and integration with the game engine.
- **LiveOps / Event Configuration Tool:**
    - **Scope:** Events, shop inventory, task parameters, and announcements.
    - **Activation:** Real-time updates vs. scheduled activation logic.

## 11.2 Installer

- **Installation Flow:** Full package vs. Small launcher + Asset streaming.
- **First Launch Experience (FTUE):** Download strategy, background animations, and loading tips.
- **Channel Specifics:** Custom requirements for Android app stores, PC launchers, etc.

## 11.3 Update Software

- **Client Update Strategy:** Forced updates, Optional updates, and Silent background updates.
- **Asset Hot-patching:** Chunking strategy, Manifest management, and Rollback mechanisms.
- **Scheduling:** Pre-download capabilities and alignment with version release cycles.

## 11.4 Internal Tools & Dashboards

- **Data Dashboard:** Visualization of KPIs (DAU, Retention, Revenue, Funnels).
- **Operations Dashboard:** Event status monitoring, reward distribution tracking, and anomaly detection.
- **GM Tools:** Customer support functions (Player account lookup, compensation, banning/unbanning).
